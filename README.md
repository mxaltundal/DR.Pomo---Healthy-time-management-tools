# DR.Pomo — Healthy Time Management Tools

A Pomodoro-based healthy time management app built with React, TypeScript, and Vite.

## 🚀 Local Development

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🌐 Free Deployment — GitHub Pages

This project is configured to automatically deploy to **GitHub Pages** for free on every push to `main`.

### One-time setup (only needed once):

1. Go to your repository on GitHub.
2. Navigate to **Settings → Pages**.
3. Under **Source**, select **GitHub Actions**.
4. Save the settings.

After that, every push to the `main` branch will automatically build and publish the app.  
Your live site will be available at:

> **https://mxaltundal.github.io/DR.Pomo---Healthy-time-management-tools/**

### Manual deploy (optional):

You can also trigger a deployment manually from the **Actions** tab in your repository by selecting the **Deploy to GitHub Pages** workflow and clicking **Run workflow**.

## 🛠️ Build

```bash
npm run build
```

The production output is generated in the `dist/` folder.
